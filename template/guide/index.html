스크린샷 가이드

  • 포블게이트 우측상단 회원가입 버튼을 클릭합니다.

  • 내/외국인을 선택하신 후, 원하는 회원가입 방식을 선택합니다.

  • 사용하시면서 인증받을 이메일과 회원정보를 입력 후, 약관 동의

  • 회원가입을 완료 후, 이메일과 계좌 인증을 꼭 진행 해주세요.

스크린샷 가이드

  • 포블게이트 우측상단 입출금 버튼을 클릭합니다.

  • 입금하실 금액을 입력 하신 후, 입금신청을 클릭합니다.

  • 입력하신 금액을 확인 하신 후, 확인 버튼을 클릭합니다.

  • 입금 주의사항에 동의한 후, 확인 버튼을 클릭해서 완료합니다.

스크린샷 가이드

  • 포블게이트 우측상단 입출금 버튼을 클릭합니다.

  • ‘자산’ 탭의 리스트중 코나A-IEO의 ‘SWAP’ 버튼을 클릭합니다.

  • ‘스왑’ 탭에서 원하시는 숫자를 입력 후, ‘스왑 신청’ 버튼을 클릭합니다.

  • 확인 창의 내용을 확인 후, ‘확인’ 버튼을 클릭합니다.

  • ‘코나’ 키워드로 검색 후, 코나코인을 선택합니다.

  • 매도 탭에서 원하시는 수량을 입력 후, 매도를 완료합니다.
